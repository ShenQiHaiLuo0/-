# 防掉盘文件复制校验器

**Anti-Disk-Drop File Copy Verifier**

一款专为科研数据、大容量存储场景设计的 PowerShell 文件复制 + SHA256 校验 + 自动修复工具。

适用于 USB 移动硬盘、NAS、网络映射盘等易掉盘环境下的数据完整性保障。

## 功能特性

| 功能 | 说明 |
|------|------|
| **一键复制** | 基于 robocopy 多线程复制，支持长路径、特殊字符、中文文件名 |
| **SHA256 校验** | 多线程并行计算 SHA256，源文件与目标文件逐一比对 |
| **自动修复** | 校验不一致的文件自动重新复制，复制后二次校验确认 |
| **掉盘重试** | 指数退避重试（5s/10s/20s/30s/60s），自动检测盘符恢复 |
| **进度可视化** | 复制、校验、修复三阶段独立进度条，实时显示 ETA |
| **磁盘空间预检** | 复制前检测目标磁盘空间，不足时提示并可选择继续；修复时逐文件检查 |
| **完整日志** | 自动生成程序日志、robocopy 日志、失败文件列表、中文校验报告 |
| **长路径支持** | 自动转换 `\\?\` 前缀，支持超过 260 字符的 Windows 路径 |
| **特殊字符兼容** | 支持 Emoji、Unicode、特殊符号（%$#&\|<>{}[]）等文件名 |

## 快速开始

### 方式一：双击运行（推荐）

直接双击 `双击运行工具.bat`，无需修改系统策略。
注：建议直接将“双击运行工具.bat”创建一个快捷方式在桌面，然后右键 属性 > 高级(D) > 勾选用管理员身份运行(R)。这样可以规避的UAC管理员身份错误提示。

### 方式二：命令行运行

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File "图形复制校验工具.ps1"
```

### 操作流程

```
选择源文件夹 → 选择目标文件夹 → 自动执行复制+校验+修复 → 生成报告
```

1. 弹出文件夹选择对话框，选择**源文件夹**
2. 弹出文件夹选择对话框，选择**目标父文件夹**（工具会自动在目标下创建源文件夹同名子目录）
3. 自动执行：
   - **复制阶段**：robocopy 多线程复制 + 进度条
   - **校验阶段**：多线程 SHA256 并行比对 + 进度条
   - **修复阶段**：自动重新复制不一致文件 + 进度条
4. 生成校验报告并自动打开

## 技术实现

### 架构设计

```
┌─────────────────────────────────────────────────────┐
│                    主程序流程                         │
├──────────┬──────────┬──────────┬────────────────────┤
│ 文件夹选择 │ 磁盘空间  │ 复制阶段  │ 校验+修复阶段       │
│ (WinForms)│ 预检     │(robocopy)│(Runspace Pool)     │
├──────────┴──────────┴──────────┴────────────────────┤
│              日志系统 (Write-Log + 缓冲)              │
├─────────────────────────────────────────────────────┤
│              报告生成 (Write-Report)                  │
└─────────────────────────────────────────────────────┘
```

### 核心技术点

| 技术 | 实现 |
|------|------|
| **多线程校验** | PowerShell Runspace Pool，线程数 = min(CPU核心数, 16) |
| **流式 SHA256** | 128KB 缓冲区 `TransformBlock`，不加载完整文件到内存 |
| **robocopy 调用** | `System.Diagnostics.Process` 替代 `Start-Process`，避免参数转义问题 |
| **长路径处理** | `\\?\` UNC 前缀 + `-LiteralPath`，支持 320+ 字符路径 |
| **BFS 遍历** | 显式队列广度优先搜索，避免 PowerShell 递归调用栈溢出 |
| **掉盘检测** | 指数退避重试 + 盘符可达性检查，USB 断开重连后自动恢复 |
| **进度条** | `\r` 回车符原地刷新，30 字符宽度，显示百分比/数量/ETA |
| **日志缓冲** | 内存缓冲 + 阈值刷盘（20条），减少磁盘 I/O |

### robocopy 参数

```
/E /COPY:DAT /DCOPY:T /R:1 /W:3 /MT:8 /NP /XO /FFT /NDL /NJS
```

| 参数 | 说明 |
|------|------|
| `/E` | 递归子目录（含空目录） |
| `/COPY:DAT` | 复制 数据+属性+时间（最安全，无权限要求） |
| `/DCOPY:T` | 复制目录时间戳 |
| `/R:1 /W:3` | 失败重试 1 次，间隔 3 秒 |
| `/MT:8` | 8 线程并行复制 |
| `/NP` | 不显示进度百分比（避免日志膨胀） |
| `/XO` | 排除旧文件（不覆盖更新的目标文件） |
| `/FFT` | 允许 2 秒时间戳误差（FAT32/NTFS 兼容） |
| `/NDL /NJS` | 不记录目录名和作业名（避免特殊字符问题） |

### 错误重试策略

| 重试次数 | 等待时间 | 累计等待 |
|---------|---------|---------|
| 第 1 次 | 5 秒 | 5 秒 |
| 第 2 次 | 10 秒 | 15 秒 |
| 第 3 次 | 20 秒 | 35 秒 |
| 第 4 次 | 30 秒 | 65 秒 |
| 第 5 次 | 60 秒 | 125 秒 |

每次重试前检查盘符是否恢复，未恢复则额外等待 10 秒。

### 可重试错误类型

- `FileNotFoundException` / `DirectoryNotFoundException` — 掉盘后路径消失
- `IOException` — 文件被锁定、I/O 设备错误
- `UnauthorizedAccessException` — 权限临时问题
- `Win32Exception` — Windows API 错误
- 30+ 种错误消息模式匹配（中英文 USB/网络/磁盘错误）

### 不可重试错误

- `PathTooLongException` — 路径超长，重试无意义
- `DiskSpaceExhausted` — 磁盘空间不足，跳过后续修复

## 环境要求

- **操作系统**：Windows 10 / 11
- **PowerShell**：5.1 或更高版本
- **依赖**：仅使用 Windows 内置组件（robocopy、.NET Framework）
- **无需管理员权限**

## 设计原则

1. **安全第一**：`/COPY:DAT` 不复制权限，避免权限问题；修复前二次校验
2. **掉盘友好**：指数退避重试 + 盘符恢复检测，适应 USB 不稳定场景
3. **大数据友好**：流式 SHA256 不占内存，BFS 遍历不怕深层目录
4. **长路径友好**：`\\?\` 前缀全局处理，支持 320+ 字符路径
5. **用户友好**：全程进度条 + ETA，中文界面，一键启动
